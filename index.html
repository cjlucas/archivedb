archivedb by cjlucas

archivedb is a program that maintains a checksum database of your media archive

Dependencies

Python (2.7+ or 3.2+) MySQL (4.1+) PyMySQL

Install

python setup.py install copy bin/archivedb to /usr/bin/ chmod +x /usr/bin/archivedb
